Sector Sentiment

Director on-market buy/sell activity grouped by GICS sector over the last 90 days. Sentiment is bullish when buy value exceeds sell value by 50%+, bearish when sell value exceeds buy value by 50%+, otherwise neutral. Click any sector card to see the top companies.

Director buy/sell activity by sector — last 90 days

Smart Money Scores

Each director-company pair is scored 0–10 on how often their on-market trades were followed by price movements in the correct direction (buys preceding rises, sells preceding falls) within 90 days. Scores are weighted by recency using exponential decay with an 18-month half-life — recent accurate trades count more than old ones.
